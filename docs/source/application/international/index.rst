
局域网中共同登录AOI的系统
=================================


1.打开C:\Program Files\DaoAI AOI路径下的launch文件
    .. image:: ./images/7.png
      :scale: 80%
  
2.可以看到在Net Work中，复制红框内的地址后在同一局域网下的另一台电脑，打开浏览器输入复制的地址进入，即可打开同一局域网内的AOI系统
    .. image:: ./images/9.png
      :scale: 80%
3.进入后会有弹窗，在弹窗内输入复制的地址，并更改端口为8000，如下图即可使用AOI系统
    .. image:: ./images/10.png
      :scale: 50%
    .. image:: ./images/11.png
      :scale: 50%
